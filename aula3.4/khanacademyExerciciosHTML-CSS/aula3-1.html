A Escola do Futuro

Curso de Desenvolvimento Web.

Trybe é a escola do futuro, faça o curso de Desenvolvendor Web
e só comece a pagar depois que conseguir um emprego que
remunere um salario de R$3.500 ou mais.

O que vamos aprender no curso ?

  • Git
  • HTML
  • CSS
  • JavaScript
  • React
  • Node
  • Express.js
  • SQL

Construir um front-end
Você aprenderá a construir um front-end a partir de especificações
de UI e UX, que se comunica com uma API.


Construir um back-end
Você aprenderá a construir um back-end, seguindo padrões de arquitetura,
conectando-se com um banco de dados e implementando uma interface REST.


Algoritmos
Você aprenderá os conceitos de Ciência da Computação, sabendo utilizar
os principais algoritmos para resolver problemas complexos.


Soft skills
Você vai trabalhar em equipe de forma colaborativa, assumindo
responsabilidades, comunicando-se de forma eficiente e sabendo conciliar interesses.


Rotinas de desenvolvimento
Você vai se ambientar com a rotina de uma pessoa desenvolvedora
de software, incluindo os processos de uma empresa de tecnologia
e as melhores práticas de desenvolvimento e de controle de qualidade de software.


Metodologias Ágeis
Você aprenderá os conceitos, artefatos e cerimônias das metodologias ágeis,
e será capaz de trabalhar com naturalidade seguindo a metodologia.
